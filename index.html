Usages:

HTML links, also known as hyperlinks, are elements that allow you to connect one web page to another. They are a fundamental part of the web and enable users to navigate between different pages and resources. Here's some important information about HTML links:

Anchor Tags

Links in HTML are created using the < code>& lt;a& gt;< / code> (anchor) element. The < code> & lt;a & gt;< /code> element requires an < code>href< / code> attribute, which specifies the URL of the target page or resource.

Text Links vs. Image Links

You can create text links using anchor tags, where the linked text is placed between the opening and closing < code>& lt;a & gt;< / code> tags. Image links are also possible by placing an < code> & lt;img& gt;< / code> element within an anchor tag.

Linking to External Resources

Links can point to external resources, such as other websites or files hosted on different servers. To link to an external resource, provide the full URL in the < code>href< / code> attribute.

Relative URLs

Relative URLs are used to link to resources within the same website. They are specified relative to the current page's URL. For example, if you have a page in a subdirectory, you can link to it using a relative URL.

Linking to Sections within a Page

You can create links that navigate to specific sections within the same page using < code>id< / code> attributes and the < code>#< / code> symbol in the < code>href< / code> attribute. This is often used for creating table of contents or one-page navigation.

Linking to Email Addresses

Links can also be used to create email links. When the user clicks on the link, their default email client opens with a pre-filled email address. This is done using the < code >mailto:< / code> scheme in the < code>href< / code> attribute.

Link Styling

Links can be styled using CSS to change their appearance when hovered over, clicked, or visited. Common styling includes changing text color, underlines, and cursor types.

Link Accessibility

It's important to ensure that links are accessible. Using descriptive link text and providing alternative text for linked images helps users understand the purpose of the link, especially for those using screen readers.

Link Target Attribute

The target attribute determines how the linked content will be displayed. For example, you can open links in a new tab using < code>target="_blank".

Link Best Practices

When designing your website, consider the placement and context of your links. Use clear and concise link text, and avoid overloading a page with too many links.

Conclusion

HTML links are an essential part of the web, enabling seamless navigation between pages and resources. By understanding how to create and use links effectively, you can enhance the user experience on your website.

Ch.8 Table

A Simple example of a table looks like this:

< table >
< tr>
< th>Day< / th>
< th>Numbers< / th>
< / tr>
< tr>
< td>monday< / td>
< td>23< / td>
< / tr>
< / table>
And that code translates to:
Day Numbers
monday 23

The HTML table element tag looks like this < table > and < / table > it also consists of more tags like this: < tr>, < th> and < td>.
tr means the table row, th means a table header and td means a table cell.
A table sometimes may include a < caption >, < colgroup>, < thead>, < tfoot> and < tobdy> tags.
